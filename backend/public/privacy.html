Política de Privacidade

Studio 59 respeita a sua privacidade. Esta política explica como tratamos os dados quando utiliza a aplicação.

Dados que recolhemos

  • Dados de conta e autenticação (ex.: email do staff).
  • Dados de eventos e pedidos associados ao seu uso da app.
  • Fotos carregadas pelo staff para eventos (quando aplicável).
  • Dados técnicos mínimos para funcionamento e segurança.

Permissões

  • Câmara: utilizada para captura de imagem em pesquisas ou operações dentro da app.
  • Galeria/Armazenamento: utilizada para selecionar e guardar fotos.

Como usamos os dados

  • Para permitir o funcionamento da aplicação e gestão de eventos.
  • Para processar pedidos e disponibilizar fotos associadas a eventos.
  • Para cumprir obrigações legais e de segurança.

Partilha de dados

Não vendemos dados pessoais. Os dados podem ser partilhados apenas quando necessário para a prestação do serviço, cumprimento legal ou segurança.

Retenção

Os dados são mantidos pelo tempo necessário para o funcionamento do serviço ou conforme exigido por lei.
